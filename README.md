# Waypoint Workflows Website — Business Presence 1.1

Static Cloudflare Pages project for `waypointworkflows.com`.

## Local development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run deploy
```

Cloudflare Pages may also deploy this directory directly. No build command is required; the project root is the output directory.

## Release notes

Business Presence 1.1 rebuild implements the approved homepage specification, production Waypoint assets, responsive navigation and layouts, metadata, Open Graph tags, full favicon set, and web app manifest.
